## 常用流程

```bash
mailrelay init
mailrelay doctor
mailrelay once
mailrelay status
mailrelay run
```

使用 `--config path` 指定配置，或设置 `MAILRELAY_CONFIG`。

退出码：`0` 成功，`1` 运行失败，`2` 配置或用法错误。
