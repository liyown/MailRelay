## 常用流程

```bash
mailrelay init
mailrelay doctor
mailrelay once
mailrelay status
mailrelay run
```

使用 `--config path` 指定配置，或设置 `MAILRELAY_CONFIG`。

启用 Web Console 时，使用 `mailrelay hash-password` 生成 Argon2id 管理员密码哈希，不要把明文密码写入配置文件或 shell 历史。

源码开发常用 Taskfile 命令：

```bash
task dev
task console-embed
task test-all
```

`task dev` 使用 Air 监听 Go、Console 和配置文件变更，自动重新构建嵌入式前端并重启服务。

退出码：`0` 成功，`1` 运行失败，`2` 配置或用法错误。
